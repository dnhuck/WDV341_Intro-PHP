CSS Preprocessor

  1. A CSS Preprocessor is responsible for allowing more a faster and more flexible way to run css as an extension.
  2. Some advantages of using scss would be that it saves time and larger projects can be split into modules making them easier to use or change if needed.
  3. Some dissadvantages of scss would be the intial set up whicb does take some time to set up. Also, it is not the easiest to learn and not all computers or text editors provide sass so it must be installed.
  4. Css preprocessors currently available:
    • Sass
    • Stylus
    • Less
    • PostCSS
    • Stylecow
  5. Every job that I looked at either required or reccomended that you understand how Sass or Ruby works.
